# Kaypic Communication
Application web de messagerie et de collaboration en temps réel développée avec ASP.NET Core et SignalR.

## Fonctionnalités
- Messagerie instantanée en temps réel
- Conversations privées et de groupe
- Notifications en direct
- Gestion de profil utilisateur
- Interface moderne et responsive

## Technologies
- ASP.NET Core MVC
- SignalR
- Entity Framework Core
- SQL Server
- HTML / CSS / JavaScript
